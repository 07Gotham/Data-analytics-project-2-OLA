# Data-analytics-project-2-OLA
Data analysis project on Ola ride dataset using Excel, SQL, Python, and Power BI to extract business insights, optimize operations, and understand customer behavior.


Project Overview
This project presents an interactive Power BI dashboard built using Ola ride data to analyze booking trends, revenue generation, and cancellation patterns.
The dashboard helps stakeholders understand operational performance and make data-driven decisions.

Dashboard Features
Key Metrics (KPIs)
Total Bookings: 103,024
Total Revenue: ₹57 Million
Success Rate: 62%

Booking Status Analysis
Majority rides are successfully completed
Significant cancellations:
Driver-side cancellations are higher than customer-side
“Driver Not Found” indicates supply issues


 Ride Volume Trends
Daily booking trends visualized
Fluctuations indicate demand variation
Helps identify peak and low-demand days


Key Visuals Used
Pie Chart → Booking status breakdown
Line Chart → Ride trends over time
KPI Cards → Business metrics
Filters → Date-based analysis


Key Insights
Only ~62% rides are successful → improvement needed
High driver cancellations → possible driver dissatisfaction
 Demand fluctuates daily → opportunity for surge pricing
 “Driver Not Found” shows supply-demand gap
Business Recommendations
Increase driver incentives during peak hours
Improve driver allocation system
Reduce cancellations via penalties or better matching
Expand driver base in high-demand areas
🛠️ Tools Used
Power BI (Dashboard)
Excel (Data Cleaning)
SQL (Data Analysis - optional if used)
